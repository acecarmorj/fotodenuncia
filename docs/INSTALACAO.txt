CLICKCIDADE - INSTALACAO
========================

Este sistema foi pensado para funcionar como o ACE:

- GitHub Pages hospeda index.html e painel.html.
- Google Apps Script recebe/envia dados.
- Google Sheets guarda as denuncias.
- Google Drive guarda as fotos.

1. Criar planilha
-----------------
Crie uma planilha no Google Drive da conta acecarmorj@gmail.com:

ClickCidade - Denuncias

O Apps Script deve ser criado a partir desta propria planilha:

Extensoes > Apps Script

Assim, o codigo cria as abas na mesma planilha em que esta sendo executado.

2. Criar pasta das fotos
------------------------
Crie uma pasta no Google Drive:

ClickCidade - Fotos

Copie o ID da pasta. O ID fica na URL:

https://drive.google.com/drive/folders/ID_DA_PASTA

3. Criar Apps Script
--------------------
No Google Apps Script, crie um projeto chamado:

ClickCidade API

Cole o conteudo de:

D:\fotodenuncia\apps-script\Code.gs

O arquivo:

D:\fotodenuncia\api.txt

deve ficar como copia de conferencia do mesmo script.

4. Configurar Script Properties
-------------------------------
Em Apps Script > Project Settings > Script Properties, adicione:

CLICKCIDADE_PHOTOS_FOLDER_ID = ID_DA_PASTA
CLICKCIDADE_PANEL_PASSWORD = senha_geral_do_painel

Opcionalmente, em vez da senha em texto, use:

CLICKCIDADE_PANEL_PASSWORD_SHA256 = hash_sha256_da_senha

Nao coloque senha real nos arquivos do GitHub.

A propriedade CLICKCIDADE_SPREADSHEET_ID nao e obrigatoria quando o script estiver dentro da propria planilha. O codigo usa a planilha ativa automaticamente.

5. Preparar planilha
--------------------
No editor do Apps Script, execute a funcao:

ClickCidade_setup

Ela cria as abas e cabecalhos necessarios.

6. Publicar Web App
-------------------
Em Apps Script:

Deploy > New deployment > Web app

Configuracao sugerida:

Execute as: Me
Who has access: Anyone

Copie a URL terminada em /exec.

7. Configurar GitHub Pages
--------------------------
Abra:

D:\fotodenuncia\assets\runtime-config.js

Cole a URL do Apps Script em API_URL.

Depois suba a pasta D:\fotodenuncia para o GitHub e ative GitHub Pages.

8. URLs finais
--------------
Cidadao:

https://usuario.github.io/repositorio/

Painel:

https://usuario.github.io/repositorio/painel.html

9. Observacoes
--------------
- GPS e camera funcionam melhor com HTTPS. Por isso o GitHub Pages e importante.
- O painel usa login simples com senha validada no Apps Script.
- A tela do cidadao nao exige cadastro.
- Nome e WhatsApp do cidadao sao opcionais.
- A localizacao por GPS e obrigatoria para enviar uma denuncia.
- A referencia do local e opcional e serve apenas como complemento.
- A foto de conclusao no painel tambem e opcional.
- O formulario pode ser instalado no celular quando o navegador mostrar a opcao.
- Antes do uso oficial, apague somente as linhas de teste da aba Denuncias e mantenha o cabecalho.
